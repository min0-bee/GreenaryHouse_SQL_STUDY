# 문제 조건
Marks로 Grade 매칭
- Students.Marks가 Grades.Min_Mark ~ Max_Mark 사이면 그 Grade
- 출력 컬럼은 3개
Name, Grade, Marks
- Grade < 8이면 이름 숨김
- Grade가 1~7인 학생들은 Name을 출력하지 말고 NULL로 출력
정렬 규칙
- 전체는 Grade 내림차순
- Grade가 8~10인 학생들끼리는 Name 오름차순(알파벳순)
- Grade가 1~7인 학생들끼리는 Marks 오름차순
(이때 Name은 NULL)
